# prompts/

Task-based instructions for AI tools. Each file in this folder is a reusable prompt for a specific design or development task.

## Usage

Reference a prompt file when starting a task that matches its scope. Example:

> "Using the instructions in `prompts/new-component.md`, generate a new list view component."

## Naming convention

`{task-type}-{subject}.md`

Examples:
- `generate-form.md`
- `review-component.md`
- `audit-spacing.md`
